## 声音角色 (3 条)

### 声音面具（Voice Mask）

- **ID**: voice-mask
- **类型**: 游戏
- **描述**: 用声音创造完全不同的人
- **标签**: 角色塑造, ★☆☆
- **能力**: 声音角色, 角色范围, 表达
- **场景**: 排练, 演出

**步骤**:

1. 选一句中性台词（如"请把那个给我"）。
2. 每人用 5 种完全不同的声音说出来——改变音高、语速、音色、共鸣位置。
3. 每种声音应暗示不同角色。进阶：两人用各自的声音角色开始场景。

**提示**: 很多新手只用"自己的声音"演戏，声音面具帮他们探索声音作为角色入口。带领者鼓励极端声音——包括"不好听"的声音，也可能是好角色。关键原则：声音变→角色变→心理变。 训练的核心能力：拓展声音表达范...

**变体**: A 用一个声音说台词，B 用同一个声音回应；每个声音配合身体姿态互相强化

### 灵魂互换/相互配音

- **ID**: 角色塑造-2
- **类型**: 游戏
- **描述**: 三人循环配音，人人都上场
- **标签**: 角色塑造, ★★☆
- **能力**: 声音角色, 协作, 身体表演
- **场景**: 排练, 演出

**步骤**:

1. 开场：A 和 B 上台开始一个场景。A 为 B 配音——B 只做身体表演不说话，A 站在一旁为 B 的角色配音。B 的身体动作 + A 的声音 = B 的角色。
2. C 加入：C 在台下为 A 配音——A 只做身体表演不说话，C 的声音从台下传来（或站在一旁）。A 的身体动作 + C 的声音 = A 的角色。
3. C 上台：C 在合适时机上台加入场景——成为第三个角色。
4. B 为 C 配音：C 上台后只做身体表演不说话——B 为 C 的角色配音。C 的身体动作 + B 的声音 = C 的角色。
5. 三人形成循环配音：A→B，C→A，B→C。每个人既是某个角色的身体，又是另一个角色的声音。没有单纯的"配音员"——三个人都在台上表演，都在为别人配音。

**提示**: 灵魂互换的独特之处在于没有人只做一件事——每个人同时是身体表演者和配音者。配音者需要精准观察被配音者的身体动作来匹配声音，表演者需要用身体给出足够的线索让配音者知道怎么配。三人之间的默契是核心—...

**变体**: 主持人随时喊"换！"——三人立刻交换配音关系；增加一个角色，形成更复杂的配音循环

### 提线木偶

- **ID**: 角色塑造-6
- **类型**: 游戏
- **描述**: 两组四人，木偶对戏
- **标签**: 角色塑造, ★★☆
- **能力**: 身体控制, 声音角色, 双人配合
- **场景**: 演出

**步骤**:

1. 两组木偶组合——A1（木偶师）操控 B1（木偶），A2（木偶师）操控 B2（木偶）。
2. 两个木偶师不能说话——只能用手势和提线动作"遥控"各自木偶的全身动作。
3. 两个木偶可以说话但不能自主移动身体——身体完全由木偶师控制，木偶用声音表达角色的想法和情绪。
4. 两个木偶在台上对戏——A1 操控 B1 的身体动作，B1 配音；A2 操控 B2 的身体动作，B2 配音。两个木偶之间的对话和身体互动形成完整的场景。
5. 声音和身体分属不同人创造了独特的戏剧张力——木偶嘴里说着"我好喜欢你"，但木偶师操控着木偶的身体不由自主地后退。

**提示**: 提线木偶的四人版本将单人操控升级为双人戏码。木偶师之间的配合变得关键——两个木偶师需要保持视觉连接，确保两个木偶的互动在物理上精准匹配（握手、拥抱、打架）。木偶演员则需要在被操控的状态下保持角色...

**变体**: 中途两组交换木偶师——新木偶师必须立刻适应新木偶的身体特质；木偶短暂夺回身体控制权——但立刻被木偶师重新操控
