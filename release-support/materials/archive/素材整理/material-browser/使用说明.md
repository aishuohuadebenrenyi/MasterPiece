# 即兴素材浏览器使用说明

## 📱 如何使用

### 在手机上查看

1. **打开HTML文件**
   - 在手机浏览器中打开 `material-browser.html` 文件
   - 或者将整个 `material-browser` 文件夹上传到云盘，通过云盘应用打开

2. **推荐方式**
   - 使用微信、QQ等应用的文件传输功能，将HTML文件发送到手机
   - 在手机上用浏览器打开该文件
   - 或者使用手机上的文件管理应用打开

### 在电脑上查看

1. 直接双击 `material-browser.html` 文件
2. 文件会在浏览器中自动打开

## 🎯 功能介绍

### 1. 总览页面
- 显示素材总数统计
- 查看类型、难度、场景分布
- 了解热门标签

### 2. 素材库页面
- **类型筛选**: 按游戏、技巧、路径等类型筛选
- **难度筛选**: 按入门、进阶、高阶等难度筛选
- **点击展开**: 点击素材卡片查看详细信息
- **加载更多**: 分页加载素材，避免一次性加载过多

### 3. 搜索页面
- 输入关键词搜索素材
- 支持搜索标题、描述、标签、能力
- 实时搜索，快速定位素材

## 📋 素材详情

每个素材卡片包含以下信息：

- **基本信息**: 标题、类型、描述
- **标签**: 素材的分类标签
- **元数据**: 人数、时长等信息
- **操作步骤**: 详细操作步骤（点击展开查看）
- **提示**: 使用提示和建议
- **变体**: 素材的变体玩法
- **注意事项**: 使用时需要注意的问题
- **训练能力**: 素材培养的能力维度
- **使用场景**: 适合的使用场景

## 🎨 设计特点

- **活力橙蓝配色**: 清晰的视觉层次
- **手机友好**: 针对手机屏幕优化
- **触摸交互**: 支持触摸操作和手势
- **快速加载**: 分页加载，避免卡顿
- **离线可用**: 所有资源本地化，无需网络

## 📂 文件结构

```
material-browser/
├── material-browser.html  (主HTML文件)
├── assets/
│   ├── materials.json     (素材数据)
│   └── materials.js       (交互逻辑)
└── _shared/
    └── fonts/             (字体文件)
```

## 💡 使用建议

1. **首次使用**: 先查看"总览"页面，了解素材分布
2. **快速查找**: 使用"搜索"功能快速定位特定素材
3. **系统学习**: 在"素材库"中按类型或难度筛选，系统学习
4. **详细查看**: 点击卡片展开，查看完整信息

## 🔄 更新素材

如果素材数据更新，需要重新生成 `materials.json` 文件：

```bash
python3 generate_browser_data.py
```
