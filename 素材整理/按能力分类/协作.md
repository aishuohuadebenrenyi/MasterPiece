### 双头专家（Two-Headed Expert）

- **ID**: two-headed-expert
- **类型**: 游戏
- **描述**: 二人合体，一字一人
- **标签**: 角色塑造, ★★★
- **能力**: 协作, 聆听, 角色
- **场景**: 排练, 演出

**步骤**:

1. 两人并排站立，合体成一个"专家"——交替，每人每次只说一个字，共同回答观众提问。
2. 主持人向专家提问（"怎么修好坏掉的自行车？"）。两人一字一字合作回答，听起来像一个完整的人。

**提示**: 双头专家是极致聆听的终极训练。无法提前计划——只知前一个人的一个字。目标不是"说出聪明的话"，而是让句子自然完成。两人都不知道走向何处，结束时却发现说出了一句完整的话——团体的奇迹。 训练的核心...

**变体**: 三人时难度指数级增加；必须用统一的情绪表达

### 灵魂互换/相互配音

- **ID**: 角色塑造-2
- **类型**: 游戏
- **描述**: 三人循环配音，人人都上场
- **标签**: 角色塑造, ★★☆
- **能力**: 声音角色, 协作, 身体表演
- **场景**: 排练, 演出

**步骤**:

1. 开场：A 和 B 上台开始一个场景。A 为 B 配音——B 只做身体表演不说话，A 站在一旁为 B 的角色配音。B 的身体动作 + A 的声音 = B 的角色。
2. C 加入：C 在台下为 A 配音——A 只做身体表演不说话，C 的声音从台下传来（或站在一旁）。A 的身体动作 + C 的声音 = A 的角色。
3. C 上台：C 在合适时机上台加入场景——成为第三个角色。
4. B 为 C 配音：C 上台后只做身体表演不说话——B 为 C 的角色配音。C 的身体动作 + B 的声音 = C 的角色。
5. 三人形成循环配音：A→B，C→A，B→C。每个人既是某个角色的身体，又是另一个角色的声音。没有单纯的"配音员"——三个人都在台上表演，都在为别人配音。

**提示**: 灵魂互换的独特之处在于没有人只做一件事——每个人同时是身体表演者和配音者。配音者需要精准观察被配音者的身体动作来匹配声音，表演者需要用身体给出足够的线索让配音者知道怎么配。三人之间的默契是核心—...

**变体**: 主持人随时喊"换！"——三人立刻交换配音关系；增加一个角色，形成更复杂的配音循环

### 三头奇葩说

- **ID**: 角色塑造-3
- **类型**: 游戏
- **描述**: 两个三头专家辩论
- **标签**: 角色塑造, ★★★
- **能力**: 协作, 极致聆听, 辩论
- **场景**: 演出

**步骤**:

1. 观众给出一个辩论话题（如"猫比狗好""早上洗澡比晚上好""外星人应该交税"）。
2. 两个三头专家——每组 3 人并排站立，合体成一个专家。正方专家 3 人交替每人每次只说一个字，共同说出正方的论点。反方专家 3 人也同样用一字接力的方式回应。
3. 两个专家交替辩论——正方三头说一个论点（3 人一字一字完成一个句子），反方三头立刻回应（3 人一字一字完成反驳）。
4. 每个三头专家的 3 人必须完美同步节奏——任何一个人的犹豫都会让整个专家的句子断裂。断裂时对方专家可以"乘胜追击"。

**提示**: 三头奇葩说是双头专家（m3-26）和三头（原 m3-91）的升级版——将一字接力从"回答问题"变成"辩论对抗"。两个专家之间的辩论节奏是关键——每个三头组内部需要极致聆听，两个三头组之间又需要辩...

**变体**: 不仅一字接力说话，每个三头专家的成员还各自控制专家的身体一部分——一人控制左臂，一人控制右臂，一人控制身体；辩论结束后观众投票——哪个三头专家说得更有道理（或更荒谬）

**注意**: 诀窍：不要试图预测下一个字，信任前一个字给出的方向。

### 小帮手

- **ID**: 角色塑造-5
- **类型**: 游戏
- **描述**: 两组四人，额外的手辅助对戏
- **标签**: 角色塑造, ★☆☆
- **能力**: 角色辅助, 协作, 身体喜剧
- **场景**: 排练, 演出

**步骤**:

1. 两组搭配——A1（演员）和 B1（小帮手）一组，A2（演员）和 B2（小帮手）一组。两个演员 A1 和 A2 在台上对戏。
2. 每个演员的双手背在身后——演员自己不能用手。各自的小帮手站在演员身后，把双手从演员腋下伸出来——帮手的手成为演员的手。
3. 两个演员正常对话和做身体动作，但所有需要手的事情——拿东西、指方向、握手、推开对方——都由各自的帮手来完成。
4. 演员和帮手配合：演员用语言和身体姿态给出指令，帮手的手精准执行。两个演员对戏时，两组手也在对戏——形成双重喜剧效果。

**提示**: 小帮手的四人版本将单人身体喜剧升级为双人对戏。两个演员在认真演戏的同时，两双手在帮（或捣乱）——四层表演同时进行。演员需要信任帮手，帮手需要敏锐观察演员的意图。这个游戏在观众面前视觉效果极佳——...

**变体**: 帮手的手故意不听话——演员想握手，帮手的手偏偏比了个中指；中途两组交换帮手——新手必须立刻适应新演员的身体
