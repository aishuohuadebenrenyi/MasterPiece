## 📖 使用说明

### 如何查找素材？

1. **按类型查找**: 如果你想找特定类型的素材（如游戏、技巧等），请查看 `按类型分类/` 文件夹
2. **按标签查找**: 如果你想找特定标签的素材（如热身破冰、场景与叙事等），请查看 `按标签分类/` 文件夹
3. **按能力查找**: 如果你想训练特定能力（如自发性、角色塑造等），请查看 `按能力分类/` 文件夹
4. **按场景查找**: 如果你想找适合特定场景的素材（如排练、演出等），请查看 `按场景分类/` 文件夹
5. **按难度查找**: 如果你想找特定难度的素材，请查看 `按难度分类/` 文件夹

### 素材信息结构

每个素材文档包含以下信息：

- **ID**: 素材唯一标识
- **类型**: 素材类型（游戏、角色、才艺、格式、主理、技巧、复盘、路径）
- **描述**: 素材一句话说明
- **标签**: 素材标签列表
- **能力**: 训练能力列表
- **场景**: 使用场景列表
- **步骤**: 详细操作步骤（如有）
- **提示**: 使用提示（如有）
- **变体**: 变体玩法（如有）
- **注意**: 注意事项（如有）
