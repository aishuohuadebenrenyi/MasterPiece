### 2026-06-28

- 落地当前个人版单素材练习记录升级：`PracticeRecord` 改为 `score / note / attachments` 主结构，练习复盘首屏收敛为 `1-10` 分评分和“本次复盘”，支持照片/短视频附件上传、保存前删除、历史附件查看与删除确认；素材详情页新增个人练习记录摘要，练习记录页支持按素材、评分区间和附件类型筛选。
- 修复练习记录升级后的移动端体验问题：素材详情页练习记录摘要按当前 `materialId` 二次过滤，避免混入其他素材记录；最近一次日期改为不换行短日期；详情页底部操作栏脱离 `root-portal` 依赖；练习记录页素材筛选改为选择器，避免全部素材 chip 横向过长和小屏重叠。
- 继续优化练习复盘和练习记录筛选：复盘评分从 `1-10` 横向 chip 改为原生滑块拖动打分，并保留 `1 / 5 / 10` 关键刻度和实时状态反馈；练习记录页素材筛选从隐式 picker 改为显式“切换”入口和底部选择面板，支持素材名称搜索、全部素材和当前选中态。
- 补齐练习记录升级后的移动端防回归：`empty-state-panel` 组件动作区支持小屏收缩和换行，素材选择弹层列表显式左对齐并缩小清除按钮，练习复盘页头部不再重复素材名而改用大号练习时长摘要，评分徽标改为垂直居中；同步沉淀卡片、半弹窗和混合字号徽标的防溢出规则，并加入轻量契约检查。
- 二次修正练习记录素材选择弹层：素材候选只展示“全部素材”和已有练习记录的素材，不再显示“暂无练习记录”项；弹层搜索框复用发现页 `search-bar` 组件，清除按钮、图标和输入节奏保持一致；候选列表改为可点击视图行，避免原生按钮布局造成左侧空白。
